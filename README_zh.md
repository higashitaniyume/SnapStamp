# SnapStamp

**SnapStamp** 是一款 Android 创意摄影应用，它可以将普通照片自动转换为具有邮票风格的艺术作品。应用基于 Jetpack Compose 和 CameraX 开发，集成了拍照、地理位置标记、个人备注和画廊管理功能。

## 应用功能

* 直接在应用内使用相机拍照。
* 自动裁切照片为邮票风格的方形图像。
* 为照片添加穿孔边框和经典邮票外观。
* 自动获取拍摄时的 GPS 定位，并将位置信息写入照片 Exif 元数据。
* 对每张邮票添加个人备注或标题文字。
* 支持导出带装饰的邮票图像，也可以保存原始干净照片。
* 提供画廊浏览功能，可翻转查看邮票详情并管理收藏。
* 支持分享到其他应用，或删除不需要的照片。

## 核心特色

* **邮票生成**：拍照后自动生成带有白色边框和穿孔边缘的邮票样式图像。
* **地理标记**：使用 FusedLocationProvider 获取当前 GPS 坐标，并将地址信息写入 Exif 数据中。
* **个人备注**：为每张照片添加自定义文字备注，记录拍摄时间、地点或心情。
* **双重导出模式**：可以选择保存已经生成邮票样式的图像，或保存原始未加装饰的照片。
* **画廊管理**：在相册中查看已保存的邮票，翻转卡片显示详细信息，并可删除或分享图像。
* **现代 UI**：采用 Jetpack Compose 实现流畅动画与简洁界面。

## 技术栈

* **Kotlin**：应用主要开发语言。
* **Jetpack Compose**：声明式 UI 框架。
* **CameraX**：用于相机拍摄和预览。
* **Coil**：高效图片加载与处理。
* **ExifInterface**：读取和写入照片元数据。
* **FusedLocationProvider**：获取精准定位信息。

## 使用说明

1. 打开应用并授予相机和位置信息权限。
2. 在相机界面拍摄照片。
3. 应用会自动裁剪图像并添加邮票边框。
4. 可选填写位置信息备注或个人说明。
5. 保存邮票到图库。
6. 在相册中浏览已保存的邮票，支持分享或删除。

## 项目结构

* `app/src/main/java/` — 应用代码和 Compose 界面。
* `app/src/main/res/` — 资源文件、图标和布局。
* `app/build.gradle.kts` — 应用模块构建配置。
* `build.gradle.kts` — 项目级 Gradle 配置。

## 适用场景

该应用适合用于旅行摄影、日记记录和创意相册制作。建议在支持 CameraX 和开启定位服务的设备上使用，以获得最佳体验。
